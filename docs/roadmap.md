# 間取りスタジオ ロードマップ（羅針盤）

**2026-07-04 ピボット済み**: 手作業CAD型 → AI生成型（経緯は `docs/pivot-generate.md`）。

オーナーの用途: ①LUNA体操教室のレッスンコース設計 ②自宅・将来の家のレイアウト/インテリア検討。
North Star: **「条件を選ぶ → 完成した部屋が複数案出てくる → 眺めて選んで、気になる所だけ直す」**。
UX原則: **「0から作らせない」**。開いた瞬間に完成案が見える。手作業は微調整のときだけ。

## 大物（人間との対話セッションで実装。ルーチンは提案・下調べまで）

1. ✅ おまかせ生成 v1（ルールベース: 1R/1LDK/2LDK/LUNAコース × 暮らし × テイスト → 3案ガチャ）2026-07-04
2. 自然文からの生成（Claude API、要オーナー承認: APIキー＝金銭コスト）
3. 生成例ギャラリーのSEOページ（「6畳 1K レイアウト」等の検索需要を拾う静的ページ量産）
4. 家具アフィリエイトリンク（生成案の使用アイテム一覧→楽天/Amazon。オーナーのASP登録待ち）
5. 平面図・断面図の出力（3Dで作る→図面PDF/PNGが出る）
6. pose-tracerレッスンとのコース図連動（プランJSON→レッスン資料PDF）
