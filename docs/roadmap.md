# 間取りスタジオ ロードマップ（日次改善ルーチンの羅針盤）

オーナーの用途: ①LUNA体操教室のレッスンコース設計 ②自宅・将来の家のレイアウト/インテリア検討。
UX原則: **「ぽちぽち押させない」**。見たまま・ドラッグで完結・開いた瞬間に使える。

## 大物（人間との対話セッションで実装。ルーチンは提案・下調べまで）

1. 平面図・断面図の出力（3Dで作る→図面PDF/PNGが出る）
2. 壁くり抜き式のドア・窓（現状は壁に重ねるパネル方式）
3. pose-tracerレッスンとのコース図連動（プランJSON→レッスン資料PDF）
4. ステーション番号（①→⑤の進行順ラベル。LUNAコース用）

## 小物（日次ルーチンが1日1つ実装してよい）

- Undo（1手戻る）
- 選択アイテムの複製ボタン
- 家具同士のスナップ/整列補助
- カタログの検索ボックス
- 使用アイテム一覧（買い物リスト風）表示
- サムネの読み込み体感改善（先読み等）
- 空プラン時のオンボーディング（サンプルプラン1タップ生成）
- モバイルのサイドバー折りたたみ改善

## 改善のネタ元（市場調査対象）

- 競合: マイホームクラウド / Floorplanner / Roomstyler / IKEA Kreativ / Planner 5D（無料枠のUXを観察）
- 「間取り アプリ 使いにくい」等の不満検索から改善種を拾う
